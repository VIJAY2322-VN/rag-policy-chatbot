# RAG Policy Chatbot

This project is a Retrieval-Augmented Generation (RAG) based chatbot that answers questions from policy documents using Large Language Models.

## Features
- Document-based question answering
- Vector search for relevant context
- LLM-powered responses

## Tech Stack
- Python
- LangChain
- FAISS / ChromaDB
- Streamlit
- Git & GitHub
